# Full-Stack Web Development Portfolio 🚀

This project is a personal portfolio designed to showcase my web development work and technical skills. It is built as a full-stack application, ensuring a dynamic experience for both users and administrators.

---

## 🛠 Tech Stack & Tools

### **Frontend**
* **HTML5 & CSS3** - Core structure and advanced styling.
* **Tailwind CSS** - Modern, utility-first CSS framework for responsive design.
* **JavaScript (ES6+)** - Logic, DOM manipulation, and asynchronous API calls.
* **Responsive Design** - Mobile-first approach for all devices.
* **Font Awesome** - Interactive vector icons.

### **Backend**
* **Node.js** - JavaScript runtime environment for the server.
* **Express.js** - Web framework for building RESTful APIs.
* **Nodemailer** - Mail server integration for sending/managing inquiries.
* **CORS** - Managing cross-origin resource sharing and security.
* **Dotenv** - Secure management of environment variables.

### **Database**
* **MongoDB** - NoSQL database for flexible data storage.
* **Mongoose** - Elegant MongoDB object modeling for Node.js.
* **MongoDB Atlas** - Cloud-based database hosting.

### **DevOps & Deployment**
* **Render** - Cloud platform for hosting the backend and frontend.
* **Git & GitHub** - Version control and source code management.

---

## 🌟 Key Features

* **Dynamic Project Gallery:** Fetches projects directly from MongoDB.
* **Secure Admin Dashboard:** Password-protected area to manage content without touching the code.
* **Full CRUD Operations:** Add, update, or delete projects in real-time.
* **Integrated Inbox:** A custom messaging system where contact form inquiries are stored and managed, allowing for direct replies.
* **Security-First:** Sensitive data is protected using environment variables.

This project is the result of continuous iterations, focusing on clean code architecture and performance optimization. It is fully responsive and tested for cross-browser compatibility.
